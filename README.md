# 🌸 Royal Fragrance - Perfume Website UI

## 📌 Project Overview

This project is a modern **Perfume E-commerce Website UI design** created using Figma.
It focuses on delivering a premium and elegant user experience for browsing and purchasing fragrances online.

## 🚀 Features

* 🌸 Premium perfume website UI
* 🏠 Home, About, Gallery, Collection pages
* 🔐 Login / Register system
* 🛍️ Product listing & product details
* ❤️ Wishlist & cart system
* 💳 Checkout & payment flow
* 👤 User profile & edit profile
* 📍 Contact page with location
